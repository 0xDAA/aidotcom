# aidotcom

A front-end mockup of [ai.com](https://ai.com), built with pure HTML & CSS — no frameworks, no JavaScript.

---

## 📸 Preview

> A clean, minimal clone of the ai.com landing page featuring a sticky glassmorphism header, full-screen hero section, and fixed parallax background.

---

## ✨ Features

- 🪟 **Glassmorphism Header** — sticky navbar with blur backdrop effect
- 🖼️ **Fixed Parallax Background** — background image stays in place while scrolling
- 🌊 **Full-screen Hero Section** — centered content with a BETA ACCESS badge
- 📋 **Mission Section** — frosted glass card with the ai.com mission statement
- 📌 **Sticky GitHub Badge** — floating widget pinned to the bottom-right corner
- 🦶 **Footer** — with copyright and policy links & Social Media links
- 🔤 **Nunito Font** — via Google Fonts
